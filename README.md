# Build

Windows

    > git clone https://github.com/zkywalker/zky
    > cd zky
    > gradlew app:assembleDebug

Linux

    $ git clone https://github.com/zkywalker/zky
    $ cd zky
    $ ./gradlew app:assembleDebug

生成的 apk 文件在 app\build\outputs\apk 目录下
